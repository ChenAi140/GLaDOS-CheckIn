## Github每天自动签到教程
1. Fork此仓库
2. 获取GLaDOS的cookie(不会自己百度)
3. 将cookie填入`Settings` -> `Secrets and variables` -> `Actions` -> `Repository secrets`中，命名一定要是`GLADOS_COOKIE`
